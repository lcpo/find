# find
Программа позволяет искать файлы через протокол ftp, по содержимому.
В приложении имеется многопоточный режим и система кеширования, она выпускается на двух языках Английский и Русский.
##Протестирована
- Pure-FTPd
- vsFTPd 2.2.2
- ProFTPD
